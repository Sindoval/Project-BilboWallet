<h1 align="left">BilboWallet</h1>

###

<p align="left">O projeto Bilbowallet é uma aplicação de carteira de controle de gastos com conversor de moedas. Com esta aplicação, os usuários podem:<br><br>- Adicionar, remover e editar um gasto.<br>- Visualizar uma tabela de gastos.<br>- Ver o total de gastos convertidos para uma moeda escolhida.</p>

###

<h2 align="left">Habilidades Demonstradas</h2>

###

<p align="left">Neste projeto, demonstrei as seguintes habilidades:<br><br>- Criar uma store Redux em aplicações React.<br>- Desenvolver reducers e actions no - Redux para gerenciar o estado global.<br>- Utilizar os hooks do Redux para manipulação e leitura do estado.<br>- Criar actions assíncronas em uma aplicação React com Redux.</p>

###

<h2 align="left">Tecnologias Utilizadas</h2>

###

<p align="left">- TypeScript: Tipagem estática para maior segurança no código.<br>- React: Biblioteca principal para construção da interface.<br>- Redux - Biblioteca de gerenciamento de estado previsível para  aplicações JavaScript/Ts, que ajuda a manter o estado centralizado.<br>- React Router Dom: Gerenciamento de rotas.<br>- Bootstrap: Estilização rápida e responsiva.<br>- Vite: Ferramenta para desenvolvimento e build.</p>

<h2 align="left">Como Clonar e Rodar o Projeto Localmente</h2>

###

<h4 align="left">Requisitos Prévidos</h4>

###

<p align="left">- Node.js na versão 18 (ou instale o nvm para gerenciar múltiplas versões de Node).<br>- npm ou yarn para gerenciamento de dependências.</p>

###

<h4 align="left">Passos para Instalação</h4>

###

<p align="left">Clone o repositório:<br><br>- git clone https://github.com/Sindoval/Project-BilboWallet<br><br>Navegue até o diretório do projeto:<br><br>- cd sd-030-b-project-trybewallet-ts<br><br>Instale as dependências:<br><br>- npm install<br><br> ou<br><br>- yarn install<br><br>Inicie o servidor de desenvolvimento:<br><br>-  run dev<br><br>ou<br><br>- yarn dev<br><br><br>Caso a aplicação não abra: <br>Acesse endereço http://localhost:5173 em seu navegador.</p>
